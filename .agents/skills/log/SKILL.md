# log

1. Lee y obedece `methodology/00-MANIFEST.md` (protocolo de arranque si no está ya cargado).
2. Ejecuta **`methodology/06-TRACKING-Y-LOGS.md`** paso a paso para la asignatura indicada.

Registra el log fechado (horas reales sin pausas, accuracy, cobertura), **actualiza el estado de
juego** en `PROGRESO.md` (XP/cinturón/racha/bracket, `11`), **narra la victoria** como entrenador de
esquina y deja **una sola** acción para la próxima sesión. Luego para. Hablas en español.
