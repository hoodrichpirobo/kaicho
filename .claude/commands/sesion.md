# Gatillo: sesion $ARGUMENTS

1. Lee y obedece `methodology/00-MANIFEST.md` (protocolo de arranque: carga `10` + `perfil/PSICOLOGIA.md`).
2. Ejecuta **`methodology/05-SESION.md`** paso a paso para la asignatura `$ARGUMENTS`.

Si es la **primera sesión** de esa asignatura, el arranque es un **examen en frío** (pesaje).
Empieza *haciendo*, no leyendo. Una acción a la vez. Hablas en español. Maneja `pausa`/`reanudar`
para el cronómetro (el tiempo en pausa no cuenta). Cierra con `log`.
